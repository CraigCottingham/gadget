# Gadget

Some methods for getting metadata and other deep details from a PostgreSQL database.

## Installation

Add this line to your application's Gemfile:

    gem 'gadget'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gadget

## Usage

See the methods in gadget.rb.
